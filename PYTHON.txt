Now create your first Flask app

Create a folder, for example:

C:\Users\USER\Desktop\myflask
Inside that folder, create a file named app.py.
Paste this code into app.py:
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "<h1>Hello from Flask!</h1><p>Your Python website is working.</p>"

if __name__ == "__main__":
    app.run(debug=True)

Open Command Prompt and go to that folder:

cd C:\Users\USER\Desktop\myflask

Run the app:

python app.py

Open your browser and visit:

http://127.0.0.1:5000

You should see:

Hello from Flask!
Your Python website is working.
